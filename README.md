                                                                Скрипт для обработки выписок.

Зайди в менеджер макросов:

    Сервис → Макросы → Организовать макросы → Python

Как запускать макрос:

    Сервис → Макросы → Выполнить макрос

Если нужно чуть точнее (по желанию)

Можно легко:

    искать TID после слова TID или ID

    брать последнее совпадение, а не первое

    логировать строки, где сумма есть, а TID не найден

Например, заменить шаблон на:

    re.compile(r'(TID|ID)[^\w]*([A-Za-z0-9\-/_]+)')
